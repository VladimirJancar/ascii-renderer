# ASCII Renderer

A C++ program that converts image files into ASCII art and prints them to console.
![Project Banner](assets/renderer_banner.png)

## Upcoming Features
- video rendering
- gif rendering
- colour support
- linux support
- printing to file
- many new command-line options
